# Classifying-JWST-HST-galaxy-mergers-with-CNNs
This project uses images from the James Webb Space Telescope (JWST) and Hubble Space Telescope (HST) to train a Convolutional Neural Network (CNN) that can tell whether a galaxy is currently merging with another or not. Merging galaxies often have distorted shapes, tidal tails, or multiple bright cores, while non-merging ones look more undisturbed.

The goal is to teach the model to spot these features automatically, making it easier for astronomers to sift through huge datasets and study how galaxy collisions influence the universeâ€™s evolution.
